# TheSoneHub အသုံးပြုနည်း လမ်းညွှန်

## 🎯 ဒီ Site ကို ဘယ်လို အသုံးပြုမလဲ

### 1️⃣ **Visitor များအတွက်**

#### **Projects Page**
- သင့် ဖန်တီးထားတဲ့ app များကို ကြည့်ရှုနိုင်ပါသည်
- Live demo link များကို နှိပ်ပြီး စမ်းသုံးနိုင်ပါသည်
- Tech stack နဲ့ features များကို လေ့လာနိုင်ပါသည်

**အကျိုးကျေးဇူး:**
- သင့် portfolio ကို သိနိုင်ပါသည်
- သင့် coding skills ကို မြင်နိုင်ပါသည်
- Inspiration ရနိုင်ပါသည်

#### **Learn Page**
- Free tutorials များ ဖတ်နိုင်ပါသည်
- Web development, AI, automation အကြောင်း သင်ယူနိုင်ပါသည်
- Myanmar language နဲ့ English ရောနှောထားပါသည်

**အကျိုးကျေးဇူး:**
- အခမဲ့ သင်ယူနိုင်ပါသည်
- Step-by-step ရှင်းပြထားပါသည်
- Real project examples များ ပါဝင်ပါသည်

#### **AI Agent Page**
- 6 မျိုး mentor modes ရွေးချယ်နိုင်ပါသည်:
  1. **TheSoneHub** - သင်နဲ့ တိုက်ရိုက် စကားပြောသလို
  2. **Life Coach** - ဘဝ ပြဿနာများ ဖြေရှင်းခြင်း
  3. **Financial Coach** - ငွေရေး ကြေးရေး အကြံပေးခြင်း
  4. **Stoic + Buddhist Philosopher** - စိတ်ဓာတ် ခိုင်မာအောင် လမ်းညွှန်ခြင်း
  5. **Learning Coach** - သင်ယူနည်း ကောင်းမွန်အောင် ကူညီခြင်း
  6. **Meta Coach** - သင်ယူနည်းကို သင်ယူခြင်း, prompt ရေးနည်းကို သင်ယူခြင်း

**အသုံးပြုနည်း:**
1. Mentor mode ရွေးပါ
2. မေးခွန်း ရေးပါ
3. AI က အဖြေပေးပါလိမ့်မည်

#### **PromptCraft Page**
- Meta prompt builder tool
- AI က မေးခွန်း မေးပြီး prompt တည်ဆောက်ပေးပါသည်
- သင် ပြင်ချင်တာ ပြောရုံပဲ

**အသုံးပြုနည်း:**
1. "New Session" (+) နှိပ်ပါ
2. AI က မေးခွန်း မေးပါလိမ့်မည်
3. သင့် လိုချင်တဲ့ prompt ရည်ရွယ်ချက် ပြောပါ
4. AI က တဖြည်းဖြည်း prompt တည်ဆောက်ပေးပါလိမ့်မည်
5. ပြုပြင်ချင်ရင် "Modify the introduction" လို့ ပြောပါ
6. ကောင်းရင် Star ⭐ ပေးပြီး template အဖြစ် သိမ်းနိုင်ပါသည်

#### **Digital Garden**
- သင့် notes, ideas များ ဖတ်နိုင်ပါသည်
- Status: Seedling 🌱 (အစ), Growing 🌿 (ဖွံ့ဖြိုး), Evergreen 🌳 (ပြီးစီး)
- Tags ဖြင့် filter လုပ်နိုင်ပါသည်

#### **Community Page**
- Discussion forum
- မေးခွန်း မေးနိုင်ပါသည်
- အခြား builders များနဲ့ ဆွေးနွေးနိုင်ပါသည်

### 2️⃣ **သင့်အနေနဲ့ (Site Owner) ဘယ်လို အသုံးပြုမလဲ**

#### **Content Management**
သင့် site မှာ content များ ထည့်ရပါမည်:

1. **Projects** - ဒီအတိုင်း အဆင်ပြေပြီးသားပါ (3 projects ရှိနေပြီ)
2. **Learn Articles** - Tutorial များ ထပ်ရေးရပါမည်
3. **Digital Garden** - Notes များ ထပ်ထည့်ရပါမည်
4. **Community** - Discussion threads များ ဖန်တီးရပါမည်

#### **AI Integration**
လက်ရှိ Google AI Studio (Gemini API) သုံးနေပါသည်:
- ပိုက်ဆံ သက်သာပါသည်
- Fast response
- Myanmar language support ကောင်းပါသည်
